JS-Test

Overview

Module tests focus on some specific aspect of a program. They are written in JavaScript, embedded in an HTML document. The js-test library provides functions like check() which compares the result of an operation with the expected value. The result (of the comparison) is recorded in the HTML document.

Tests can be grouped in sections, which give test modules some kind of structure. Sections and tests are numbered automatically.

Implementation Details

Collecting Changes

When a test is run, the following state variables change:

  • sectionSummary.testsPerformed += 1
  • moduleSummary.testsPerformed += 1

When a test succeeds, the following state variables change:

  • testContainer.testStatus = "success"
  • sectionSummary.testsSucceeded += 1
  • moduleSummary.testSucceeded += 1
  • testContainer.testStatus = "fail"
  • testSection.tests
  • testSection.testStatus = "fail"
